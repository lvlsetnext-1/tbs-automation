TB&S Automation — How to Use & Quick Wins

We’ve implemented an automation that connects dispatch → billing → invoicing using Google Sheets + Apps Script. Dispatchers enter loads once; normalized data flows to billing and invoices in a few clicks.

Potential ROI (benchmarks you can refine):
  • Save 5–10 minutes per load (no re-entry).
  • Reduce invoice errors by 30–60% via protected totals & validation.
  • Cut invoice cycle time by 25–40%; possible DSO reduction 1–2 days.

Problem we solved: duplicate data entry across dispatch, tracking, and invoicing caused delays and errors.

What you get: a formula-driven Export (real-time), a TB&S menu for one-click actions, and protected invoice templates. See the steps below (also shown inside the TB&S/About workbook).

Dispatcher — Daily
  1. Open Shared Dispatch → Working and update loads.
  2. Export updates automatically via formulas.
  3. Audit: check Export has Y: “Last Updated” and Z1 timestamp.
Tracker / Biller — Happy Path
  1. In the TB&S sheet’s menu: Clone Template (targets any prior month's template).
  2. Click Sync From Dispatch to fetch the latest dispatcher data.
  3. If needed, ask Admin to run Apply Validations on affected ranges.
  4. Click Create Invoices, then review protected totals/headers and export/share.
